Creating scenes

1. Save Scene

The Save Scene option allows you to save the current scene you're working on.

- CTRL + S to save or manually in the options menu.

2. Load Scene

The Load Scene option allows you to load a previously saved scene. You will need to drag and drop the .json file from the stages/saves folder into the file selection area.

If in V2 of our engine, just double click in the json file

3. New Scene

The New Scene option allows you to create an empty scene in your project. This will create a new scene file within the stages/saves folder, where all your scenes are stored.

- CTRL + N to create a new scene.
