Getting Entities

You can retrieve entities created in the engine by calling the following method from the ecs class:

BY CODE

ecs.get_component<TransformComponent>(0)->position = { 0.0f, 10.0f, 0.0f };

ecs.get_component<TransformComponent>(5)->rotation = glm::vec3(0.0f, (float)time, 0.0f);

This allows already created entities to be modified by code. The getEntity function retrieves an entity by its ID. The ID is shown in the Identifier component.

BY LUA

-- Example Lua Script for Transform component local sx, sy, sz = getScale() setScale(1, 2, 1) local yRotation = 0 function update(dt) local x, y, z = getPosition() --setPosition(x + dt, 0, 0) yRotation = yRotation + dt * 5 setRotation(0, yRotation, 0) end

With this method you can edit all the entities that you allready have in the scene.
