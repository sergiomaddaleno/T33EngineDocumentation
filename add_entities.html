Adding entities to scene

There are several ways to create entities in T33Engine:

  • From the Engine Interface: You can create entities directly from the engineâ€™s UI, in the tools option.
  • By Code (RenderSystem Functions): T33Engine provides functions within the RenderSystem that allow you to cretae different entity types.
  • By Code (Manual Implementation): If you prefer full control, you can manually write the necessary code to define the entities, with your desired components.

Using RenderSystem class functions:

You just have to call the setup() mehtod one time before creating entities.

      auto mario_mesh = std::make_unique();
      
      RenderSystem render_system;
      ECSManager ecs;
      Shader shader_tex;

      if (!mario_mesh->loadOBJ("../stages/assets/obj/Mario.obj", "../stages/assets/textures/mario")) {
          return -1;
      }
      
      auto j = ecs.create_entity();
      ecs.add_component_type();
      ecs.add_component_type();
      
      ecs.add_component(j);
      ecs.add_component(j);
      ecs.get_component(j)->type_ = RT_Mesh;
      ecs.get_component(j)->mesh_ = std::move(mario_mesh);
      ecs.get_component(j)->position = glm::vec3{ 0.0f, 0.0f, 0.0f };
      ecs.get_component(j)->scale = glm::vec3{ 0.2f, 0.2f, 0.2f };
      
      initShader(shader_tex, "../stages/shaders/basic/texture.vs", "../stages/shaders/basic/texture.fs");
      render_system.init(ecs);
      render_system.shader = shader_tex;
